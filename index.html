OVI - GIT y GITHUB

Sistema de control de versiones

Objetivo General

Dar a conocer la herramienta de control de versiones para trabajo en equipo de grades y pequeños proyectos

Objetivos Especificos

  • Contextualizar sobre GIT y GITHUB su funcionamiento e integración.
  • Enfatizar la importancia de usar el sistema de control de versiones GIT y GITHUB.
  • Implementación de a herramienta necesarias para usar GIT y GITHUB.

Que es Git

Es un sistema de control de versiones distribuido que se encarga de almacenar el versionamiento del proyecto o producto en un conjunto de instantáneas de un mini sistema de archivos. Git maneja tres estados principales para organizar los archivos

Que es GitHub

Es una plataforma para el desarrollo colaborativo de software que permite el alojamiento de proyectos implementando el sistema de control de versione GIT

Que es GitHub Desktop

Es una aplicación que permite mediante una interfaz grafica realizar las operaciones de control de versionamiento del proyecto, es decir, permite realizar todas operaciones de consola mediante la interfaz grafica, además de esto permite ver los cambio realizados comparar los archivos locales con los que están almacenados en al nube es decir la plataforma GITHUB, crear un repositorio, clonarlo, ver las ramificaciones del proyecto.
